# Brand Assets

This folder contains the core visual identity files for Cheat Editor Manager Tool.

## Core Brand Assets

- Primary logo: `primary-logo.png`
- Secondary logo: `secondary-logo.png`
- Wordmark (text-only logo): `wordmark.png`
- Logomark (icon/symbol-only logo): `logomark.png`
- App icon: `app-icon.png` and `app-icon.ico`
- Watermark: `watermark-brand.png`

## Runtime Files Used By App

- `app-icon.ico`
- `wordmark-360.png`
- `logo-header.png`
- `mark-48.png`

## Regenerate Assets

Run:

```bash
python assets/generate_brand_assets.py
```
