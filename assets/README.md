# Brand Assets

Generated retro-console branding for Cheat Editor Manager Tool.

Files:
- `app-icon.ico`: Windows executable icon
- `icon-256.png`: app icon image
- `mark-48.png`: compact UI mark
- `logo-header.png`: header and hero logo
- `logo-wide.png`: wide wordmark
- `watermark-ui.png`: compact translucent watermark for the app UI
- `watermark.png`: large translucent watermark panel art
- `generate_brand_assets.py`: source generator
